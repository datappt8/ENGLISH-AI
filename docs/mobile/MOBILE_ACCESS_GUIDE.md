## 🔍 故障排查

### 问题1: 手机无法访问
**检查**:
- 手机和电脑是否在同一WiFi
- 电脑防火墙是否允许端口5000和5173
- IP地址是否正确

**解决**:
```bash
# Windows - 查看IP
ipconfig

# 允许防火墙
# 控制面板 → Windows Defender 防火墙 → 允许应用
```

### 问题2: 显示"Network Error"
**检查**:
- 后端服务是否运行
- 前端服务是否运行
- 是否已登录

**解决**:
- 刷新页面
- 清除浏览器缓存
- 重新登录

### 问题3: 按钮点不了
**检查**:
- 是否在游戏画面中点击NPC
- 对话框是否弹出

**解决**:
- 点击游戏中的NPC角色
- 不是点击底部的"开始对话"按钮
