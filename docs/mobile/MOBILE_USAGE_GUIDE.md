# 📱 手机端使用指南

## 🎮 基本操作

### 1. 访问游戏
- 手机浏览器访问: `http://192.168.1.245:5173`
- 确保手机和电脑在同一WiFi

### 2. 横屏游戏
- 将手机横屏以获得最佳体验
- 点击右上角 "⛶" 按钮进入全屏模式
- 点击右上角 "✕" 按钮退出全屏

### 3. 移动角色
- **滑动屏幕**即可移动角色
  - 向上滑动 = 向上移动
  - 向下滑动 = 向下移动
  - 向左滑动 = 向左移动
  - 向右滑动 = 向右移动

### 4. 与NPC对话
- **点击NPC角色**（村长喵喵、面包师、图书管理员）
- 对话框会自动弹出

## 🎤 语音对话功能

### 如何使用语音回复

1. **点击NPC开始对话**
   - 对话框会显示NPC的消息

2. **选择语音回复**
   - 点击 "🎤 语音回复" 按钮
   - 浏览器会请求麦克风权限

3. **允许麦克风权限**
   - 首次使用需要允许浏览器访问麦克风
   - Safari: 点击"允许"
   - Chrome: 点击"允许"

4. **开始说话**
   - 看到麦克风图标后开始说话
   - 用英语回复NPC的问题
   - 说完后会自动识别并发送

5. **AI回复**
   - AI会分析你的语音内容
   - 给出英语回复
   - 自动朗读AI的回复（如果支持）

### 其他对话选项

- **💬 继续对话**: 文字输入继续对话
- **❓ 询问任务**: 询问当前任务信息
- **👋 结束对话**: 关闭对话框

## 🔧 故障排查

### 语音功能不工作

**问题1: 浏览器不支持**
- 使用Chrome或Safari浏览器
- 更新到最新版本

**问题2: 没有麦克风权限**
- 检查浏览器设置
- Safari: 设置 → Safari → 网站设置 → 麦克风
- Chrome: 设置 → 隐私和安全 → 网站设置 → 麦克风

**问题3: 语音识别失败**
- 确保网络连接正常
- 说话清晰，避免环境噪音
- 使用英语进行对话

### 画面问题

**问题1: 看不到游戏画布**
- 刷新页面（下拉刷新）
- 清除浏览器缓存
- 重新登录

**问题2: 画面太小**
- 横屏手机
- 点击右上角 "⛶" 进入全屏

**问题3: 触摸不灵敏**
- 确保不在对话框上滑动
- 在游戏画面空白处滑动
- 避免快速连续滑动

## 💡 使用技巧

### 最佳实践
1. **横屏游戏**: 获得更大的视野
2. **全屏模式**: 沉浸式体验
3. **语音练习**: 提高英语口语
4. **慢速滑动**: 更精确的移动控制

### 语音对话技巧
1. **清晰发音**: 说话清晰，语速适中
2. **完整句子**: 使用完整的英语句子
3. **等待识别**: 说完后等待1-2秒
4. **重试机制**: 识别失败可以重新点击语音按钮
