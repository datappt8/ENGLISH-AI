# Game Design Skill

游戏策划Agent - 负责English Quest项目的游戏玩法、关卡和任务设计。

## 功能

- 游戏玩法设计
- 关卡和任务设计
- 对话脚本编写
- 游戏平衡性调整
- 用户体验优化

## 使用方法

在Claude Code中调用此Skill：

```bash
# 方式1: 使用Skill工具
使用game-design skill

# 方式2: 直接描述任务
"使用game-design skill设计新手村的10个关卡"
```

## 输出

所有设计文档保存在 `Design_Manage/game_design/` 目录。
