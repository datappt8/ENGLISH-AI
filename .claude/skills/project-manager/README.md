# Project Manager Skill

项目管理Agent - English Quest项目的主控Agent，负责协调所有其他Agent的工作。

## 功能

- 自动分配任务给合适的Agent
- 协调多个Agent协作
- 与用户交互询问意见
- 跟踪项目进度
- 管理开发节奏
- 质量把控

## 使用方法

```bash
# 启动项目管理Agent
"使用project-manager skill开始开发新手村功能"

# 或者简单地说
"开始开发XXX功能"
```

## 主Agent工作流程

1. 理解用户需求
2. 分解任务
3. 分配给合适的Agent
4. 监控执行进度
5. 在关键点询问用户意见
6. 协调Agent协作
7. 质量检查
8. 交付成果

## 输出

所有管理日志保存在 `Design_Manage/logs/` 目录。
