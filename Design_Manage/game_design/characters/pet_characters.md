# 新手村拟人化宠物角色设计

**项目**: English Quest MVP
**版本**: 1.0.0
**设计日期**: 2026-01-31
**设计者**: Character Design Agent

## 🎨 设计理念

### 为什么选择宠物拟人化？

1. **超高吸引力** - 年轻人普遍喜爱萌宠
2. **无版权风险** - 动物形象属于公共领域
3. **性格鲜明** - 动物天然带有性格标签
4. **易于识别** - 每个角色有明确的视觉特征
5. **表情包潜力** - 容易制作表情包传播
6. **市场验证** - 动物森友会、疯狂动物城等成功案例

### 设计原则

- **可爱优先** - Q版风格，2-3头身比例
- **保留特征** - 明显的动物特征（耳朵、尾巴、毛色）
- **性格外化** - 外貌和动作体现性格
- **情绪丰富** - 10+种表情系统
- **动作生动** - 每个角色有独特的动物动作

## 🐱 角色 001: 村长喵喵 (Chief Meow)

### 基本信息

- **角色ID**: `village_chief_meow`
- **动物**: 橘猫
- **年龄**: 看起来像慈祥的老猫（实际年龄不详）
- **职业**: 新手村村长
- **性格**: 温和、智慧、有点懒洋洋但很靠谱

### 外貌设计

**毛色**:
- 主色：橘色（#FF9966）
- 辅色：白色（#FFFFFF）
- 橘白相间的条纹图案

**服装**:
- 深蓝色村长背心
- 金色徽章（村长标志）
- 小圆眼镜（金色镜框）

**身体特征**:
- 圆滚滚的身材（略微发福）
- 短小的四肢
- 粗壮的尾巴（经常摆动）
- 总是眯着眼睛微笑

**配色方案**:
```
主色调: 橘色 #FF9966
辅助色: 白色 #FFFFFF
服装色: 深蓝 #2C3E50
点缀色: 金色 #FFD700
```

### 表情系统

1. **默认表情** - 眯眼微笑 😊
2. **欢迎** - 举起小爪子，眼睛弯成月牙
3. **思考** - 用爪子摸下巴，眼镜反光
4. **开心** - 尾巴高高竖起，"喵~"
5. **打哈欠** - 张大嘴，露出小舌头
6. **伸懒腰** - 前爪伸直，弓起背
7. **满意** - 舔爪子，眼睛眯成一条线
8. **鼓励** - 竖起大拇指（爪子）
9. **困倦** - 眼皮下垂，打瞌睡
10. **惊讶** - 眼睛睁大，胡须竖起

### 动作设计

**待机动作**:
- 慢慢摇尾巴
- 偶尔舔爪子
- 打哈欠伸懒腰

**对话动作**:
- 说话时耳朵会动
- 开心时"喵~"一声
- 点头时眼镜会晃动

**特殊动作**:
- 打招呼：举起右爪挥手
- 同意：点头，尾巴摇摆
- 思考：用爪子推眼镜

### 声音设计

- **音色**: 温和的中年男性声音
- **语速**: 不紧不慢
- **特点**: 句尾偶尔带"喵~"
- **音效**:
  - 打呼噜声 "purr purr"
  - 喵叫声 "meow~"
  - 打哈欠声 "nya~"

### 台词风格

```
"Hello there, meow~ Welcome to Starter Village!"
"*purr purr* That's wonderful!"
"*stretches* Let me think about that..."
"Excellent, meow! You're doing great!"
```

## 🐶 角色 002: 柴犬小柴 (Shiba)

### 基本信息

- **角色ID**: `villager_shiba`
- **动物**: 柴犬
- **年龄**: 年轻活泼（相当于人类20多岁）
- **职业**: 村庄向导、冒险者
- **性格**: 超级热情、话多、好奇心强、有点傻乎乎

### 外貌设计

**毛色**:
- 主色：黄褐色（#D4A574）
- 辅色：白色（#FFFFFF）
- 经典柴犬配色

**服装**:
- 红色围巾（标志性）
- 休闲冒险装（棕色背心）
- 小背包

**身体特征**:
- 永远笑眯眯的表情（柴犬式微笑）
- 尾巴卷曲，总是摇个不停
- 三角形的耳朵（会根据情绪变化）
- 圆圆的脸颊

**配色方案**:
```
主色调: 黄褐色 #D4A574
辅助色: 白色 #FFFFFF
服装色: 红色 #E74C3C, 棕色 #8B4513
点缀色: 黑色 #2C2C2C（鼻子、眼睛）
```

### 表情系统

1. **默认表情** - 柴犬式微笑 😄
2. **超级开心** - 眼睛眯成线，嘴巴大张
3. **兴奋** - 吐舌头，尾巴摇得飞快
4. **好奇** - 歪头，耳朵竖起
5. **疑惑** - 歪头另一边，眼睛转圈
6. **惊讶** - 耳朵竖直，眼睛睁大
7. **得意** - 挺胸，尾巴高高翘起
8. **害羞** - 用爪子挠头，脸红
9. **累了** - 吐舌头喘气
10. **撒娇** - 眼睛水汪汪，尾巴轻摇

### 动作设计

**待机动作**:
- 不停摇尾巴
- 左右张望
- 原地小跳

**对话动作**:
- 说话时歪头
- 兴奋时跳起来
- 同意时疯狂点头

**特殊动作**:
- 打招呼：前爪举起，尾巴摇摆
- 带路：转圈圈，然后指方向
- 开心：原地转三圈

### 声音设计

- **音色**: 活泼清脆的声音
- **语速**: 较快，充满活力
- **特点**: 经常说"Woof!"
- **音效**:
  - 汪汪叫 "Woof woof!"
  - 喘气声 "huff huff"
  - 兴奋的呜呜声 "whine~"

### 台词风格

```
"Woof woof! Hi hi! I'm Shiba!"
"*tail wagging excitedly* This is SO exciting!"
"*jumps up and down* Let's go! Let's go!"
"*tilts head* Really? That's amazing!"
```

### 基本信息

- **角色ID**: `baker_bacon`
- **动物**: 粉红小猪
- **年龄**: 憨厚的中年猪（相当于人类30多岁）
- **职业**: 面包师

- 白色厨师帽
- 白色围裙（上面有面包图案）
- 棕色衬衫

**身体特征**:
- 圆滚滚的身材
- 大大的猪鼻子（会动）
- 小小的眼睛（笑起来眯成线）
- 卷曲的小尾巴

**配色方案**:
```
主色调: 粉红色 #FFB6C1
辅助色: 白色 #FFFFFF
服装色: 棕色 #8B4513
点缀色: 深粉色 #FF69B4（鼻子）
```

### 表情系统

1. **默认表情** - 憨厚微笑 😊
2. **开心** - 鼻子哼哼，眼睛眯起
3. **思考** - 用蹄子挠头
4. **推荐美食** - 眼睛发光，流口水
5. **满意** - 双手叉腰，点头
6. **害羞** - 脸红，用蹄子捂脸
7. **惊讶** - 鼻子张大，耳朵竖起
8. **专注** - 认真看着，鼻子抽动
9. **疲惫** - 擦汗，喘气
10. **自豪** - 挺胸，双手抱胸

### 动作设计

**待机动作**:
- 擦拭围裙
- 闻闻空气（嗅觉灵敏）
- 整理厨师帽

**对话动作**:
- 说话时鼻子会动
- 思考时用蹄子挠头
- 推荐时眼睛发光

**特殊动作**:
- 欢迎：张开双臂
- 递东西：双手捧着
- 开心：原地小跳，哼哼

### 声音设计

- **音色**: 憨厚温和的男性声音
- **语速**: 不快不慢
- **特点**: 偶尔"哼哼"
- **音效**:
  - 哼哼声 "Oink oink~"
  - 嗅闻声 "sniff sniff"
  - 满意的哼声 "hmm~"

### 台词风格

```
"Oink oink~ Welcome to my bakery!"
"*sniff sniff* Fresh bread, just baked!"
"*scratches head with hoof* Let me think..."
"*happy oink* Come back anytime!"
```

## 🐰 角色 004: 兔子露露 (Lulu)

### 基本信息

- **角色ID**: `villager_lulu`
- **动物**: 白兔
- **年龄**: 年轻可爱（相当于人类18-20岁）
- **职业**: 村民
- **性格**: 温柔、善良、有点害羞、爱哭但很坚强

### 外貌设计

**毛色**:
- 主色：雪白色（#FFFFFF）
- 耳朵内侧：粉色（#FFB6D9）

**服装**:
- 粉色连衣裙
- 白色蝴蝶结（头上）
- 小花饰品

**身体特征**:
- 大大的红眼睛（容易泪汪汪）
- 粉色的长耳朵（会根据情绪变化）
- 小巧的身材
- 毛茸茸的小尾巴

**配色方案**:
```
主色调: 白色 #FFFFFF
辅助色: 粉色 #FFB6D9
服装色: 粉色 #FFC0CB
点缀色: 红色 #FF6B9D（眼睛）
```

### 表情系统

1. **默认表情** - 温柔微笑 😊
2. **害羞** - 脸红，用爪子捂脸
3. **哭泣** - 眼泪汪汪，耳朵垂下
4. **担心** - 耳朵垂下，咬嘴唇
5. **开心** - 眼睛弯成月牙，蹦跳
6. **惊讶** - 耳朵竖直，眼睛睁大
7. **感激** - 双手合十，眼含泪光
8. **紧张** - 耳朵抖动，小声说话
9. **放心** - 松一口气，耳朵放松
10. **兴奋** - 蹦蹦跳跳，耳朵摇摆

### 动作设计

**待机动作**:
- 轻轻摇晃
- 整理耳朵
- 小幅度蹦跳

**对话动作**:
- 紧张时耳朵垂下
- 害羞时捂脸
- 开心时蹦跳

**特殊动作**:
- 着急：跺脚
- 感谢：鞠躬
- 开心：转圈圈

### 声音设计

- **音色**: 温柔甜美的女性声音
- **语速**: 较慢，有点颤抖
- **特点**: 容易哽咽
- **音效**:
  - 啾啾声 "squeak~"
  - 抽泣声 "sob sob"
  - 开心的叫声 "yay~"

### 台词风格

```
"*sob sob* I... I lost my cat..."
"*sniff* Can you help me? Please?"
"*teary eyes* Thank you so much!"
"*happy bunny hops* You're so kind!"
```

## 🎭 角色互动设计

### 角色关系

```
村长喵喵 (领导者)
    ↓
小柴 (活跃分子) ←→ 培根 (老实人)
    ↓                    ↓
露露 (需要帮助的人)
```

### 互动场景

**场景1: 村长介绍小柴**
- 喵喵: "这是小柴，他会带你参观村庄，meow~"
- 小柴: "Woof! 让我来！我最喜欢带新朋友参观了！"

**场景2: 小柴推荐培根的面包店**
- 小柴: "培根的面包超好吃！我每天都去！"
- 培根: "*oink* 谢谢小柴，你是我最好的顾客！"

**场景3: 露露感谢玩家**
- 露露: "谢谢你帮我找到Mimi..."
- 喵喵: "你做得很好，meow~ 这就是我们村庄的精神！"
